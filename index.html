Summary: Latitude vs. x Factor

This project involved analyzing weather changes as one gets closer to the equator. The main objective of the the project was to visualize the relationship of differerent cities latitude versus other weather factors such as temperature,wind speed, humidity, and cloudiness.

The goal of the project was accompolished through making successive API calls from the OpenWeatherMapAPI and randomly selecting a minimum of 500 cities across the world.Cities latitude versus other factors were plotted using libraries such as matplotlib

This site provides the visualizations as well as the data used in the Analysis.Each plot shows the relationships and a brief description of the trends.
